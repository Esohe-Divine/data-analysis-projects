# Cosmetics Analysis

### The goal is to analyze the ingredients and products using excel functions to gain insights that a chemist or product developer can use.

### Data Source: Data camp
This data contains, product names, Ingredients, skin preference and prices.

### Tools 
Excel - Data cleaning, analysis and visualization [Download Here](https;//microsoft.com)

### Data Cleaning/Preparation
In the intial data preparation phase, we performed the following tasks:
1. Data loading and inspection,
2. Handling missing values,
3. Data cleaning and formatting.

### Exploratory Data Analysis
EDA involved exploring the cosmetics data to answer key questions, such as:
- Which products contain a specific known problematic ingredient and what is the count of products containing each?
- What are the top 10 most frequently used ingredients across all cosmetic products in the dataset?
- What is the percentage distribution of products across these categories?

### Data Analysis

- if formulas, Pivot tables
